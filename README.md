# Discord Mobile Chat Layout

Esercizio di replica del layout mobile di **Discord**. Il focus del progetto è l'implementazione di una struttura complessa utilizzando le proprietà di allineamento di CSS.



## 🎯 Obiettivo
Testare le conoscenze su:
* Struttura semantica HTML.
* Allineamenti con **Flexbox**.
* Gestione degli spazi (padding, margin).
* Organizzazione di una UI multi-colonna su mobile.

---

## 🛠️ Tecnologie Utilizzate
* **HTML5**
* **CSS3**
